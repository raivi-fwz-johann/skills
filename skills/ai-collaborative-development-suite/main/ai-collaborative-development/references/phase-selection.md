# 阶段选择触发器

[设计阶段]
触发词: 方案, 范围, 约束, 风险, 验收标准, 架构选择

[画图阶段]
触发词: 架构图, 流程图, 活动图, 序列图, 状态图, 生成图表

[实现阶段]
触发词: 开始编码, 落接口, 补测试, 按步推进

[检视阶段]
触发词: 审查代码, AI 初检, 找死代码, 评估边界

[验证阶段]
触发词: 验证, 验收, 测试结果, 覆盖率, 确认通过

[恢复/继续流转触发器]
IF 当前状态 == "实现进行中":
  IF 未检视 -> [检视阶段]
  IF 未验证 -> [验证阶段]
  IF 验证已通过 -> [实现阶段] (开启下一小步)
